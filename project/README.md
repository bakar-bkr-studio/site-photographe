# BKR Studio - Site Web Professionnel

Site web officiel de BKR Studio, photographe et vidéaste professionnel basé à Meaux et Reims.

## 🎯 À propos

BKR Studio propose des services de photographie et vidéographie professionnels :
- Mariages et événements
- Portraits et famille
- Corporate et entreprise
- Sport et événementiel

## 🚀 Technologies utilisées

- **Frontend** : React 18 + TypeScript
- **Styling** : Tailwind CSS
- **Animations** : Framer Motion
- **Routing** : React Router DOM
- **Icons** : Lucide React
- **Build** : Vite
- **Backend** : Supabase Edge Functions
- **Email** : Resend API
- **Déploiement** : Netlify

## 📱 Fonctionnalités

- ✅ Design responsive et moderne
- ✅ Portfolio interactif avec galerie photos/vidéos
- ✅ Système de réservation en ligne
- ✅ Formulaire de contact fonctionnel
- ✅ Blog intégré
- ✅ SEO optimisé
- ✅ Analytics Google intégré
- ✅ Gestion des cookies (RGPD)

## 🛠️ Installation et développement

```bash
# Cloner le repository
git clone https://github.com/[VOTRE-USERNAME]/bkr-studio-website.git

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build pour la production
npm run build
```
